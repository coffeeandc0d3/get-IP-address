# get-IP-address
simple c program to output IP

# [Installation]

 mkdir build
 
 cd build

 git clone https://github.com/coffeeandc0d3/get-IP-address

 cd get-IP-address

 sudo chmod u+x display-ip.sh

# Run: 
 ./display-ip.sh
